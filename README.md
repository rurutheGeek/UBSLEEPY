# UBSLEEPY
main.pyが実行されるpythonファイルです  
すべてのファイルは文字コード*UTF-8*で記述してください  
pythonの書式は*PEP8*に準拠します  

##ディレクトリ  
*bot_module  
main.pyで呼び出したい自作モジュールを入れる  
*resource
ポケモン図鑑や画像ファイルなどのリソースファイルを入れる
*document
botから呼び出して投稿したい文章が書かれたテキストファイルを入れる  
discordで使用できるマークアップも使用可能  
*save
レポートなどのユーザーデータ,一時保存ファイルを入れる  
*中身はbotが更新するため,gitignoreで共有無視*  
*log
ログファイルを入れる  
*中身はbotが更新するため,gitignoreで共有無視*  
*temp
.gitignoreファイルで無視する設定にしている  
開発中に作成したが共有したくないファイルを入れる  

##その他のファイル
*.gitignore
gitで管理されないファイルを指定するファイル  
*config.json
botの初期設定データなどが記録されているファイル  
*README.md
このリポジトリの説明書
